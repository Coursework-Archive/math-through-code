# Calculus

This directory contains handwritten coursework, computational explorations,
and reference material for single-variable calculus.

## Structure
- handwritten/: Exam-aligned handwritten solutions (PDF scans)
- notebooks/: Python notebooks for intuition and verification
- reference/: Definitions, theorems, and recurring patterns
- exam-prep/: Condensed review material for exams

All submitted work is handwritten and prepared using a non-graphing
scientific calculator to match exam conditions.
